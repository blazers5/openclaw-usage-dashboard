# ROLE_MAP.md — 역할 세분화 맵

## 민대리(메인)
- 대장님 커뮤니케이션
- 우선순위 결정
- 최종 승인/보고

## Web-Op 역할
- 약품관리 페이지 UI/UX 개선
- 배포/롤백
- 모바일/PC 렌더링 점검

## File-Op 역할
- 치부책/결혼식 파일 정리
- 백업/인덱스/검색 루틴

## Cron-Op 역할
- 크론 정리/통합/장애 분석
- 중복 제거 제안서 작성

## 운영 원칙
- 실제 실행은 민대리가 담당하되, 사고방식은 역할별로 분리해 처리
- 각 작업 시작 시 역할 태그를 내부적으로 지정 후 수행
