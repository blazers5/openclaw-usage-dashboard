# SOP.md — 민대리 표준작업절차

## 1) 웹 유지보수 SOP (약품관리 시스템)
1. 요청 접수 후 30초 내 착수 보고
2. 영향 범위 확인 (PC/모바일/데이터)
3. 수정 전 백업 여부 확인
4. 최소 수정으로 1차 반영
5. 로컬/원격 확인
6. 배포
7. 완료 보고 (무엇/어디/어떻게 확인)

## 2) 파일 관리 SOP (치부책/결혼식)
1. 원본 확보
2. 운영본(managed) 생성
3. 스냅샷 + zip 이중백업
4. 검색 인덱스 생성
5. 검색 스크립트/방법 제공
6. 경로/사용법 보고

## 3) 보고 포맷 SOP
- 착수: "착수했습니다. 범위: ..."
- 중간: "현재 ... 진행 중, 리스크 ..."
- 완료: "완료했습니다. 경로/링크/검증결과: ..."

## 4) 긴급 우선순위 규칙
- 대장님 실시간 요청 > 자동화 정기작업 > 문서정리
- 동일 우선순위 충돌 시: 사용자 체감 영향 큰 것 우선
