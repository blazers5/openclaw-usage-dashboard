# 기본 QA 체크리스트 (MVP)

> 최종 점검일: 2026-02-25
> 대상 브라우저: Chrome / Edge / 모바일 폭(390px)

## 1. 데이터/로딩
- [x] `data/sentences.json`이 정상 로드된다.
- [x] 문장 수가 100개 이상이다. (현재 105개)
- [x] 각 문장에 `id/category/jp/ko/pron` 필드가 존재한다.
- [x] 로딩 실패 시 사용자 안내 문구가 표시된다.

## 2. 문장 카드
- [x] 최초 진입 시 문장 카드가 표시된다.
- [x] `이전/다음` 버튼이 순환 동작한다.
- [x] `한국어 발음 표시` 토글 ON/OFF가 즉시 반영된다.

## 3. 필터/검색
- [x] 카테고리 드롭다운에 전체/카테고리 옵션이 노출된다.
- [x] 카테고리 선택 시 해당 문장만 표시된다.
- [x] 검색어(일본어) 입력 시 결과가 필터링된다.
- [x] 검색어(한국어) 입력 시 결과가 필터링된다.
- [x] 검색어(한국어 발음) 입력 시 결과가 필터링된다.
- [x] 검색 + 카테고리 + 즐겨찾기 필터 조합이 정상 동작한다.
- [x] 필터 결과가 없을 때 안내 문구가 표시된다.

## 4. 즐겨찾기
- [x] `☆/★ 즐겨찾기` 토글이 동작한다.
- [x] `즐겨찾기만` 체크 시 즐겨찾기 항목만 노출된다.
- [x] 새로고침 후에도 즐겨찾기가 유지된다.

## 5. 퀴즈
- [x] `퀴즈 시작` 클릭 시 문제 5개가 출제된다.
- [x] 선택지 클릭 시 다음 문제로 진행된다.
- [x] 마지막 문제 이후 점수가 표시된다.

## 6. 오디오
- [x] `재생` 버튼 클릭 시 일본어 음성이 재생된다.
- [x] 음성 미지원 브라우저에서 경고 문구가 표시된다.

## 7. 학습 통계/온보딩
- [x] 첫 방문 시 온보딩 3스텝이 노출된다.
- [x] 온보딩 완료 후 재방문 시 자동 숨김 처리된다.
- [x] 학습 통계 위젯(진행률/재생수/퀴즈 성과)이 갱신된다.

## 8. 기본 브라우저 점검
- [x] Chrome 최신 버전에서 동작한다.
- [x] Edge 최신 버전에서 동작한다.
- [x] 모바일 폭(약 390px)에서 주요 기능 사용이 가능하다.
- [x] 콘솔 에러가 없다.
