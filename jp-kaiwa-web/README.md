# jp-kaiwa-web

일본어 회화 트레이너 MVP 웹앱입니다. 정적 파일 기반으로 동작하며, 브라우저에서 바로 실행 가능합니다.

## 현재 MVP 기능

- 문장 카드 학습 (일본어 원문/한국어 뜻/한국어 발음)
- 이전/다음 순환 네비게이션
- 일본어 TTS 재생 (`ja-JP`, 0.8x/1.0x/1.2x)
- 즐겨찾기 저장/해제 (localStorage 유지)
- 즐겨찾기 전용 보기 필터
- 카테고리 필터
- 일본어/한국어/발음 검색
- 퀴즈 v1 (4지선다, 점수 표시, 현재 필터 범위 기준 출제)
- 학습 통계 위젯 (학습 문장/진행률/재생수/퀴즈 성과)
- 첫 방문 온보딩 3스텝
- 반응형 레이아웃 (모바일 390px 대응)

## 데이터 스키마

`data/sentences.json`

```json
{
  "sentences": [
    {
      "id": "j001",
      "category": "인사",
      "jp": "おはようございます",
      "ko": "안녕하세요(아침)",
      "pron": "오하요 고자이마스"
    }
  ]
}
```

- 필수 필드: `id`, `category`, `jp`, `ko`, `pron`
- 현재 데이터: **105문장**

## 실행 방법

### 1) 로컬 서버 실행 (권장)

프로젝트 루트에서 정적 서버를 실행하세요.

```bash
# 예시 (Node)
npx serve .
```

브라우저에서 안내된 주소(예: `http://localhost:3000`)로 접속 후, `jp-kaiwa-web/` 경로를 열어 사용합니다.

### 2) 파일 직접 열기

`index.html`을 직접 열어도 기본 UI는 동작하지만, 브라우저 보안정책에 따라 JSON 로딩이 막힐 수 있습니다. 가능하면 로컬 서버 사용을 권장합니다.

## 디렉토리 구조

- `index.html`: 메인 화면 + 온보딩 모달 마크업
- `css/style.css`: 스타일(반응형, 통계 위젯, 온보딩 포함)
- `js/app.js`: 앱 로직(필터/퀴즈/TTS/통계/온보딩/localStorage)
- `data/sentences.json`: 학습 문장 데이터
- `PLAN.md`: 제품/스프린트 계획
- `TICKETS.md`: 개발 티켓
- `QA_CHECKLIST.md`: 점검표 및 최신 QA 결과

## 저장되는 로컬 상태

- `jp.fav`: 즐겨찾기 ID 목록
- `jp.settings`: 발음 표시 여부, TTS 재생속도
- `jp.progress`: 학습 통계(학습 문장, 재생수, 퀴즈 성과)
- `jp.onboardingDone`: 온보딩 완료 여부
