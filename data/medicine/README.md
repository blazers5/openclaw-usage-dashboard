# Medicine Data (정리본)

## 파일 목록
- `medicine_inventory_source_2026-02-19.xlsx`  
  원본 약품 데이터 엑셀 파일
- `medicine_inventory_webapp_source_2026-02-19.html`  
  전달받은 웹앱 원본 HTML
- `..\..\med-inventory.html`  
  현재 작업 중인 약품관리 시스템 파일

## 백업 위치
- `backups/medicine/`

## 사용 가이드
1. 엑셀 원본은 `medicine_inventory_source_2026-02-19.xlsx`를 기준으로 유지
2. 웹앱 원본은 변경하지 않고 보관
3. 실제 수정/개발은 `med-inventory.html`에서 진행
4. 배포 전에는 `backups/medicine/`에 날짜 백업 생성 권장
