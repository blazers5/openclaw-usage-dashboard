# TEAM-ROLES.md

최종 업데이트: 2026-02-25 17:48 (Asia/Seoul)

## 프로젝트 공통 운영
- 메인 총괄: 민대리 (`agent:main:main`)
- 원칙: 역할별 분업 + 결과 통합 보고 + 한국어 존댓말

## 서브 에이전트 분담 내역

### 1) veteran-planner
- 역할: 베테랑 기획자 (PM/기획 리드)
- 주요 책임:
  - MVP 범위 정의
  - 화면 IA 설계
  - 2주 스프린트 계획
  - 우선순위 백로그 관리

### 2) elite-researcher
- 역할: 국가대표급 조사원 (리서치 리드)
- 주요 책임:
  - 학습 방식/표기법 조사
  - 저작권/라이선스 리스크 점검
  - 데이터/오디오 출처 검증
  - 근거 링크 기반 리포트 작성

### 3) world-class-dev
- 역할: 세계 최상급 개발자 (기술 리드)
- 주요 책임:
  - MVP 기술 설계
  - 데이터 모델/아키텍처 정리
  - 코드 스켈레톤/테스트 전략 수립
  - 구현 품질 기준 제시

### 4) world-class-designer
- 역할: 세계 최정상급 디자이너 (UX/UI 리드)
- 주요 책임:
  - UX 문제정의
  - 사용자 플로우/화면 구조
  - 디자인 시스템(타이포/컬러/컴포넌트)
  - 접근성/개발 전달 스펙

### 5) overnight-dev-until-0800
- 역할: 야간 개발 가속 전담
- 주요 책임:
  - 야간 고도화 작업 지속 수행
  - 카테고리/검색/데이터 확장/문서화
  - 오전 보고 준비

### 6) jp-kaiwa-fastlane
- 역할: 고속 구현 전담 (배포 직전 준비)
- 주요 책임:
  - 온보딩/통계 위젯 등 고도화
  - QA 반영/문서 정합성 마감
  - 배포 직전 상태 보고

## 적용 프로젝트
- `jp-kaiwa-web` (일본어 회화 학습 웹앱)

## 운영 메모
- 역할 분담은 프로젝트 성격에 맞춰 추가/변경 가능
- 배포 전에는 반드시 메인 총괄(민대리) 최종 점검 수행
